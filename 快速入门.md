# 🎯 新手入门 - 一图看懂

如果 `README.md` 太长，看这个就够。

## 步骤 1️⃣：装 Node.js

去 https://nodejs.org 下载 LTS 版本，一路下一步安装。

## 步骤 2️⃣：打开命令行

**Windows：**
- 在 `simple-ai` 文件夹里按 `Shift + 右键` → 选「在此处打开 PowerShell」

**macOS：**
- 打开「终端」(Terminal)
- 拖动 `simple-ai` 文件夹到终端窗口，前面打 `cd ` (有空格)，回车

## 步骤 3️⃣：复制粘贴这两行

```
npm install
npm run dev
```

第一行可能要等 1-3 分钟。第二行成功后你会看到一个网址 `http://localhost:5173/`。

## 步骤 4️⃣：浏览器打开网址

把 `http://localhost:5173/` 粘贴到 Chrome / Edge / Safari 地址栏。

## 步骤 5️⃣：填 API

点首页那个大卡片「**第一步：配置 API**」。

填进去：

- **API 地址**：点预设按钮自动填，推荐 "DeepSeek"
- **API Key**：去 https://platform.deepseek.com/api_keys 申请一个

点「**保存并开始使用**」。

## 完成 ✅

现在回首页点任何卡片就能用了。

---

## 想关掉应用？

命令行里按 `Ctrl + C`。

## 想再次启动？

进入 `simple-ai` 文件夹，打开命令行，输入：

```
npm run dev
```

（之后只需要这一行，不用再 `npm install`）

## 想在手机 / 桌面上像 App 一样用？

用 Chrome 打开应用 → 地址栏右边「安装」图标 → 点它。
现在桌面 / 主屏幕就有图标了。
